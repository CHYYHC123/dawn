# DAWN

## 1.背景定制化

### 基础

- 上传本地图片作为背景
- 全屏背景/模糊效果/深色遮罩
- 背景亮度调节

// 背景可以扩展成一个“主题商店”概念，后面再做。

### 高级

- 动态背景：视频背景 / CSS 粒子效果 / Canvas 动态背景
- 背景轮播（按天、按小时、随机）
- 可考虑调用一些免费 API：
  - Unsplash 随机风景
  - 自定义主题（科技、自然、赛博朋克等）
  - 每天自动更新

## 2. 快捷功能组件
- 搜索框（可切换百度/谷歌/必应）
- 天气
- 时间 / 日历
- 每日一句（支持自定义来源）
- 专注模式（隐藏干扰内容）
